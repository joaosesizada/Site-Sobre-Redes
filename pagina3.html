O que é Apache

Apache é um software de servidor web de código aberto e gratuito que é responsável por aceitar solicitações de clientes (como navegadores web) e enviar de volta as respostas apropriadas, como páginas HTML.

O que é Tomcat

Tomcat é um servidor de aplicações de código aberto que executa aplicações Java servlets e JavaServer Pages (JSP). É desenvolvido pela Apache Software Foundation e é amplamente utilizado para servir aplicações web Java.

O que é PAN, LAN, MAN, WAN

  • PAN (Personal Area Network): Uma rede de curto alcance que é usada para conectar dispositivos pessoais, como smartphones, tablets e laptops.
  • LAN (Local Area Network): Uma rede que cobre uma pequena área geográfica, como um escritório ou edifício. É usada para conectar computadores e outros dispositivos dentro dessa área.
  • MAN (Metropolitan Area Network): Uma rede que cobre uma área geográfica maior que uma LAN, como uma cidade ou campus universitário.
  • WAN (Wide Area Network): Uma rede que cobre uma grande área geográfica, como um país ou continente. A Internet é o maior exemplo de WAN.
